# Custom Skills

These skills capture Charlie's default coding preferences in a small, reusable form.

## Skills

- `coding-style-core/` — App code structure, naming, abstraction boundaries
- `tailwind-style/` — Tailwind layout/styling preferences
- `component-system-style/` — Shared component and primitive-wrapper philosophy
- `refactor-boundaries/` — How invasive changes should be during normal feature work
- `backend-style-t3-trpc/` — Backend organization for T3/tRPC routers, schemas, queries, mutations, and router-specific utils
